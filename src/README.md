# src/

Код бота и его runtime.

Основные модули:
- `index.js`: входная точка, подключение плагинов, цикл запуска/рестарта.
- `planner.js`: принятие решений, вызовы LLM, исполнение действий.
- `llm_client.js`: клиент Ollama, выбор модели, JSON-формат ответов.
- `skills.js`: набор инструментов/действий, которыми управляет LLM.
- `perception.js`: сбор контекста и снимка окружения.
- `chat.js`: обработка команд и фидбека из чата.
- `memory_store.js`: память о мире и игроках.
- `reflexes.js`: быстрые рефлексы (угрозы, еда).
- `observer.js`: события мира и бантер.
- `training_store.js`: правила обучения, фидбек и приоритеты.
